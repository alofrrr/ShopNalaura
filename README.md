## 🚀 Sobre o Projeto

Projeto realizado para processo seletivo. 

## 📋 Pré-requisitos

```
Node.js
```

## 🔧 Como abrir o projeto

Abrir o arquivo com o Visual Studio Code  <br/>
Abrir o terminal (PowerShell) <br/>
Executar o comando:
```
npm install
```

E por fim, executar o comando: 

```
npm start
```
